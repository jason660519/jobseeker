功能特色

智能路由

根據搜尋條件自動選擇最適合的求職網站，提高搜尋效率

多格式下載

支援 CSV 和 JSON 格式下載，方便後續處理和分析

API 文檔

開發者友好的 RESTful API

搜尋職位 API

POST /api/v2/search
請求參數：
{
  "query": "軟體工程師",
  "location": "台北",
  "results_wanted": 20,
  "hours_old": 168,
  "page": 1,
  "per_page": 20
}
回應格式：
{
  "success": true,
  "total_jobs": 150,
  "jobs": [...],
  "pagination": {
    "page": 1,
    "per_page": 20,
    "total": 150,
    "total_pages": 8,
    "has_next": true,
    "has_prev": false
  }
}
